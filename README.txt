Tutorial de instalação

1º Instale o python, disponível na pasta _redist;
2º Instale o webdriver manager, disponível na pasta _redist;
3º Execute o scrip main.py através do prompt:
	>>> cd path to main.py
	>>> python main.py
4º Após inicialização do painel Cabot, feche o sistema;
5º Crie um atalho na área de trabalho.
Pronto!
